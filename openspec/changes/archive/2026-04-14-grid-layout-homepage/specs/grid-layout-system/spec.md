## ADDED Requirements

### Requirement: 固定单元格的网格布局
仪表板SHALL显示具有固定单元格尺寸的网格布局，单元格宽度100px × 高度100px，单元格之间间距20px。

#### Scenario: 网格以正确尺寸渲染
- **WHEN** 仪表板页面加载
- **THEN** 网格背景显示均匀间隔的单元格，尺寸为100px × 100px，单元格之间间距20px

### Requirement: 网格坐标系
系统SHALL使用网格坐标系，其中每个单元格由（列，行）坐标标识，从左上角的（0,0）开始。

#### Scenario: 位置映射到网格坐标
- **WHEN** 卡片定位在像素坐标（x, y）
- **THEN** 系统将其映射到最近的网格单元格坐标（列, 行），其中列 = floor(x/120)，行 = floor(y/120)

### Requirement: 网格对齐
卡片在定位或调整大小时SHALL对齐到网格单元格边界，与最近的网格线对齐。

#### Scenario: 拖拽期间卡片对齐
- **WHEN** 用户将卡片拖拽到网格线附近
- **THEN** 卡片自动将其边缘对齐到网格线

### Requirement: 网格背景可见性
系统SHALL提供视觉网格背景，使用微妙的线条指示单元格边界。

#### Scenario: 网格线显示
- **WHEN** 仪表板处于编辑模式
- **THEN** 可见微弱的网格线显示单元格边界

### Requirement: 网格边界
网格SHALL具有定义的边界，防止卡片被拖拽到可见网格区域之外。

#### Scenario: 卡片在网格内受约束
- **WHEN** 用户尝试将卡片拖拽超出网格边界
- **THEN** 卡片在网格边缘停止
