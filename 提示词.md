# 角色

你是一个前端专家

# 任务

实现一个网格布局的首页，支持动态添加卡片到网格中。

# 网格设计

- 首页使用网格设计
- 每个格子宽 100px，高 100px，上下间隔 20px
- 用户可以向网格中添加卡片
- 卡片吸附在网格上
- 可通过鼠标拖动调整卡片的布局，包括定位、尺寸

# 卡片设计

- 每个卡片都有不同的功能，如日历、时钟、待办等
- 每个卡片都有自己独立的运行环境（js、css 隔离）
- 卡片支持通过代码文本动态创建
- 创建好的卡片可以添加到网格中
- 卡片尺寸根据占用的格子数量决定，例如：横向占用1个格子，则卡片宽度 100px；横向占用2个格子，则卡片宽度 220px（100px x 2 + 20px）

# 设计风格

- 白色风格为主
- 页面中的元素（按钮、卡片等）应用圆角，圆角大小统一 16px
- 卡片适当应用高斯模糊、阴影等效果以增加高级感
- 应用一些简单的动效，如果拖拽卡片动效，动效时长不要超过 0.3s

# 角色

你是一个前端专家

# 任务

优化网格布局设计

# 相关文件

网格布局文件：frontend/src/components/Dashboard/Dashboard.tsx

# 网格逻辑

- 横向网格一共 12 个，总宽 1200px，整个网格水平居中
- 竖向网格数量可动态变化，最多不超过 24 个，最高不超过 2400px，自上而下布局
- 当拖拽卡片到视窗底部边缘时，页面可以缓慢往下滚动，卡片拖拽结束时，吸附在最近的网格中
- 添加卡片时，遵守从上到下、从左到右的原则，查找能够足够渲染卡片的网格空间
- 添加卡片时，若整个网格都不足以放下卡片，则提示用户“首页空间不足”
- 网格线正常状态下隐藏，拖拽过程中显示
